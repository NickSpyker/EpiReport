# EpiReport
An interface to epitech's coding style checker

## Installation
```shell
sudo curl -L https://github.com/NickSpyker/epireport/releases/download/v1.0/epireport > ./epireport && sudo chmod +x ./epireport && sudo mv ./epireport /usr/bin
```
